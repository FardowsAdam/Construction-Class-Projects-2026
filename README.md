# Construction-Class-Projects-2026

This repository is used for organizing all student projects for the Construction Course (2026).

Clone the repository:
git clone <repo-url>
Create a branch for your submission:
git checkout -b your-name-submission
Add your project inside your folder:
/your_name_project/
Commit your changes:
git add .
git commit -m "Add my project submission"
git push origin your-name-submission
Create a Pull Request on GitHub.
