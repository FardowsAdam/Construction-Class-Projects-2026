# Construction Class Projects 2026

This repository is used for organizing all student projects for the Construction Course (2026).

---

## 📌 Instructions

### 1. Clone the repository
```bash
git clone <repo-url>
```
### 2. Create a branch for your submission
```bash
git checkout -b project-name
```
### 3. Add your project

Create a folder using your project name  and add your project inside it:
```bash
/project_name/
```

### 4. Commit your changes
```bash
git add .
git commit -m "Add my project "
```
### 5. Push your branch
```bash
git push origin project-name
```
